# 工业3D模型展示

这是一个基于 Web 的工业 3D 模型展示页面，使用 Three.js 实现模型渲染，使用 Tailwind CSS 构建界面。  
页面适合用于工业设备、零部件、机械结构等模型的网页端展示与预览。

## 当前版本特点

当前版本基于已确认的显示效果进行整理，重点如下：

- 保持当前已确认的模型显示效果
- 页面初始不加载默认模型
- 页面打开后显示上传提示
- 支持上传本地 3D 模型
- 支持暂停旋转 / 继续旋转
- 控制面板中的颜色选择放在最上方
- 模型颜色支持四种预设
- 上传后自动居中模型
- 上传后自动调整模型大小，避免模型显示过小

## 支持格式

当前支持以下格式：

- GLTF（`.gltf`）
- GLB（`.glb`）
- OBJ（`.obj`）
- STL（`.stl`）

## 控制说明

### 鼠标操作

- 左键拖动：旋转模型
- 滚轮：缩放模型
- 右键拖动：平移视图

### 顶部按钮

- 重置视图：恢复默认视角与缩放状态
- 暂停旋转：暂停或继续模型自动旋转
- 上传模型：打开上传窗口并导入本地模型

### 控制面板

控制面板包含以下项目：

1. 模型颜色
2. 旋转速度
3. 缩放比例
4. 环境光照

## 颜色预设

当前版本提供以下四种颜色：

- 蓝灰色
- 蓝色
- 橙色
- 绿色

## 页面行为说明

### 默认状态

页面打开后不再显示示例模型，而是在画布区域显示提示信息，引导用户上传自己的模型文件。

### 模型上传后

模型上传成功后，页面会自动执行以下步骤：

1. 读取模型文件
2. 将模型添加到场景中
3. 计算模型尺寸
4. 自动居中模型
5. 自动放大或缩小模型到合适显示大小
6. 自动调整相机位置，确保模型尽量完整显示

### 不同格式处理方式

- GLTF / GLB：尽量保留原始材质效果
- OBJ / STL：使用页面内置材质显示

## 使用方法

### 直接打开

可以直接使用浏览器打开 `index.html` 运行页面。

### 本地静态服务运行

如果本地直接打开时遇到浏览器限制，建议通过本地静态服务器运行。

例如使用 Python：

```bash
python -m http.server 8000
```

然后在浏览器访问：

```bash
http://localhost:8000
```

## 注意事项

- 建议使用现代浏览器，如 Chrome、Edge、Firefox
- 页面依赖 WebGL，浏览器需支持 3D 图形渲染
- 大体积模型可能会有较长加载时间
- 不同建模软件导出的模型单位可能不同，但当前版本已做自动尺寸适配
- 如果 OBJ 模型缺少材质信息，则会使用默认材质显示
- 上传模型仅在当前页面会话有效，刷新后需要重新上传
